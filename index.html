Lightweight Security Testing Agent

Полнофункциональная облегченная версия агента для тестирования безопасности с RESTful API управления. Быстрое развертывание, Docker контейнеризация и полная изоляция.

Ключевые возможности платформы

Облегченная версия с фокусом на простоту развертывания и управления

🚀

Быстрое развертывание

Одна команда для развертывания на Windows и Linux с помощью Docker и скриптов.

🔐

JWT Аутентификация

Безопасный доступ с использованием access/refresh токенов для управления API.

📊

RESTful API

Полноценный API для управления сессиями и инстансами с автоматической Swagger/OpenAPI документацией.

🐳

Docker Контейнеризация

Полная изоляция сервисов через Docker Compose, production-ready конфигурация.

💾

PostgreSQL & Redis

Надежное хранение сессий и пользователей в PostgreSQL, а также кеширование и очереди через Redis.

📈

Мониторинг

Встроенные health checks, логирование и метрики состояния инстанса.
